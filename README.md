# 🧩 Segmentación Estratégica de Consumidores y Perfiles Socioeconómicos (ENAHO 2024)

**Período:** Febrero 2026

## 🎯 Descripción del Proyecto
Desarrollo de un modelo de inteligencia comercial mediante técnicas de Clustering 
para segmentar la población peruana, utilizando los microdatos de la Encuesta 
Nacional de Hogares (INEI - Trimestre 1, 2024). El proyecto transformó datos 
crudos en segmentos accionables para el análisis de mercado y la toma de 
decisiones estratégicas.

## 🔍 Logros y Metodología

### 📦 Gestión y Consolidación de Big Data
- Integré y procesé múltiples fuentes de datos (Vivienda, Empleo, Ingresos y Sumaria)
- Herramienta: Python (Pandas), logrando una visión 360° del perfil del ciudadano/hogar

### 🏗️ Ingeniería de Variables de Negocio
- Diseñé indicadores clave como el **Ingreso Per Cápita** y la **Condición de Actividad Económica**
- Fundamentales para identificar el poder adquisitivo y el comportamiento del consumidor

### 🤖 Modelado Analítico con Foco en Negocio
- Implementé un algoritmo de **K-Means**, optimizado mediante el Método del Codo y el Coeficiente de Silueta
- Resultado: segmentación robusta en **3 clusters diferenciados**

### 💡 Identificación de Oportunidades
- Clasificación de grupos según vulnerabilidad y capacidad de gasto
- Facilitó la identificación de nichos de mercado y personalización de estrategias según estratos geográficos
